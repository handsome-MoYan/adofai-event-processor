1. 🎯 猫咪基本工作流程

🐾 详细猫步

  1. 挑选要抓的文件：点击四处逛逛…或将 .adofai / .txt 文件拖进猫窝窗口。
    • 🐟 支持格式：.adofai、.txt、.json
    • 👃 自动闻闻编码（UTF-8、UTF-8-SIG、GBK、Shift-JIS、ASCII）
  2. 设置放鱼干的篮子：点击自动埋便便使用输入文件所在篮子，或四处逛逛选其他位置。
  3. 给小鱼干取名：输入文件名（不要刺哦），留空则使用 原文件名_new.adofai。
  4. 添加猫薄荷条件：用关键词树添加大猫薄荷或小猫薄荷（支持神奇猫抓板正则表达式）。
    • 🐱 大猫薄荷：主要抓老鼠条件（同一猫窝内是"或"逻辑）
    • 🐱 小猫薄荷：附加条件（与大猫薄荷是"与"逻辑）
  5. 选择扑击模式：从6种抓老鼠模式中选择（详见抓老鼠模式详解）。
  6. 开始扑击！点击开始扑击！或按 Ctrl+Shift+P。
🐾 注意：如果输出鱼干已存在，工具会自动创建唯一鱼干名（如 鱼干(1).adofai），不会覆盖原来的宝藏！

3. 🎯 抓老鼠模式详解

扑击模式说明猫咪使用场景
🗑️ 丢掉有猫薄荷的行 丢掉匹配至少一个大猫薄荷的行（猫窝内"或"逻辑） 快速丢掉特定老鼠类型
🐟 只留带猫薄荷的行 仅保留匹配至少一个大猫薄荷的行 分离特定老鼠用于分析
🚫 丢掉全是猫薄荷的行 丢掉同时匹配所有大猫薄荷及其小猫薄荷的行 多条件精确丢掉
✅ 只留全是猫薄荷的行 仅保留同时匹配所有大猫薄荷及其小猫薄荷的行 提取高度特定的老鼠
🎨 修改带猫薄荷的行 将匹配行的整行内容换成指定猫毛 批量替换老鼠参数
🔄 换掉猫薄荷里的字 在匹配行中查找并替换猫毛（非整行替换） 更新特定值同时保留猫窝结构
🐱 重要猫胡子警告：使用小猫薄荷时，它们与所属大猫薄荷形成"与"逻辑。不同大猫薄荷之间形成"或"逻辑。记住：小猫听大猫的话！

4. 🧰 猫咪工具箱

4.1 🧹 聪明猫猫大扫除

自动清理 ADOFAI 鱼干中的常见问题，让猫窝保持整洁：

🐾 清理空猫爪印 (EmptyMoveCleaner)

删除引用不存在装饰球标签的 MoveDecorations 事件。

🐱 示例场景：
- 你有针对 "deco1" 的 MoveDecorations 猫爪印
- 但不存在标签为 "deco1" 的装饰球
- 本工具会擦掉这些没用的猫爪印

🎾 清理不要的装饰球 (UnusedDecorationCleaner)

删除从未被其他事件引用的装饰球事件（AddDecoration、AddText 等）。

  • 分析来自 MoveDecorations、SetText、EmitParticle 等的引用
  • 处理 hitboxDecoTag 递归引用（猫球套猫球）
  • 支持只看看不动手模式，先预览要丢掉的内容
  • 修改前自动藏好鱼干备份
🐾 最佳猫咪实践：始终先启用"只看看不动手（乖猫咪）"模式查看将要丢掉的内容，确认无误后再执行真正的大扫除！

4.2 ✨ 魔法猫爪特效 (VFX制作)

为视频录制创建优化版本的鱼干关卡：

制作模式猫咪用途会丢掉什么
🐱 主猫 / 谱子 (Foreground) 用于游玩的谱面 后猫装饰球、CustomBackground、HallOfMirrors
🐱 后猫 / 录制 (Background) 用于仅后猫录制 游戏事件（SetPlanetRotation、ColorTrack 等）、主猫装饰球

🌟 超级魔法猫爪 (Video VFX Pro)

基于深度值和遮罩设置的智能猫咪过滤：

  • 🐱 主猫模式：保留负深度装饰球，根据遮罩设置过滤
  • 🐱 后猫模式：保留非负深度装饰球
  • 智能处理 Flash 和 SetFilterAdvanced 事件
  • 尊重 maskingType 和 maskingDepth 设置（猫咪很懂礼貌）

6. 🥫 使用猫罐头（预设）

🐱 创建 presets.json（猫罐头仓库）

在应用程序目录创建名为 presets.json 的猫罐头仓库：

{
  "常用老鼠": {
    "旋转球": ["SetSpeed", "Twirl"],
    "颜色球": ["ColorTrack", "RecolorTrack", "AnimateTrack"]
  },
  "装饰球": {
    "所有装饰球": ["AddDecoration", "AddText", "AddParticle", "AddObject"],
    "移动猫爪": ["MoveDecorations", "MoveTrack"]
  }
}

🐾 应用猫罐头

  • 🐱 大猫咪开吃：将猫罐头值添加为顶层猫薄荷
  • 🐱 小猫咪开吃：将猫罐头值添加为选中的大猫薄荷的子项
  • 🐱 猫猫大餐开动：在猫咪工具箱标签页批量应用整个豪华猫窝

8. 🎓 聪明猫咪进阶技巧

🔮 神奇猫抓板模式（正则表达式）

使用神奇猫抓板进行复杂匹配：

🐱 示例猫抓板：
"eventType":\s*"SetSpeed"     - 匹配 SetSpeed 老鼠（任意猫步空格）
"floor":\s*(100|101|102)      - 匹配特定楼层号
"angle":\s*\d{3,}             - 匹配3位及以上数字的角度

⚡ 飞毛腿优化（性能优化）

  • 🚀 飞毛腿模式：对大鱼干（>10MB）禁用进度更新，扑击速度提升10倍！
  • 🐟 批量处理：使用豪华猫窝在一次扑击中应用多个猫薄荷
  • 🌐 编码选择：ADOFAI 鱼干使用 UTF-8-SIG 确保正确处理 BOM

🐋 处理大鱼干

  1. 鱼干大于10MB时启用飞毛腿模式
  2. 使用有限行数闻一闻检查设置
  3. 如遇内存问题，考虑分块扑击
  4. 启用"保存猫爪印"跟踪扑击历史

9. 🆘 猫咪常见问题

问题猫咪解决方案
🚫 "这个文件不能吃（不是文本）" 确保鱼干不是二进制格式。ADOFAI 鱼干应为 UTF-8 编码的文本。
😵 处理时出现 UnicodeDecodeError 试试其他猫语编码（ADOFAI 用 UTF-8-SIG）或使用自动闻闻。
✂️ 闻一闻显示内容被剪掉 在闻一闻设置中增加"最大闻一闻行数/猫毛数"。
🥫 猫罐头无法加载 检查 presets.json 是否为有效 JSON 且位于应用程序目录。
🎾 工具箱清理找不到装饰球 确保鱼干使用标准 ADOFAI 格式，包含 actions/decorations 猫爬架。
💥 大鱼干出现内存错误 启用飞毛腿模式或分小块扑击。
🎨 毛色主题无法应用 某些毛色可能需要重启猫窝。试试切换两次或重启应用。

10. 🔧 猫语技术细节

📦 鱼干结构

ADOFAI 鱼干采用 JSON Lines 猫语格式：

{ "pathData": "...", "settings": {...} }
{ "floor": 1, "eventType": "SetSpeed", "speedType": "Bpm", "beatsPerMinute": 100 }
{ "floor": 2, "eventType": "Twirl" }
...

🧠 猫咪处理逻辑

  • 逐行扑击：每行独立评估（一只老鼠一只老鼠抓）
  • 猫抓板缓存：编译的猫抓板会被缓存以提高扑击速度
  • 内存高效：大鱼干使用流式扑击处理
  • 线程安全：后台扑击不阻塞猫窝界面
